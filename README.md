# Sorting Visualizer
### Dependencies
- Windows 11
- Powershell
- Git
- Vcpkg
- Nuget
- MSBuild

### Setting Up Project
1. Clone this repo.
```
git clone --recurse-submodules https://github.com/collindang88/SortingVisualizer.git
```

2. Enter the repo.
```
cd SortingVisualizer
```

3. Run the build script.
```
.\build.ps1
```

### Running Project
Open solution in Visual Studio. Then run in Debug mode.
